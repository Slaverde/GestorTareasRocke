# Instrucciones para agregar el logo de Rocket

## Pasos para agregar la imagen del logo:

1. **Coloca la imagen del logo** en la carpeta `static/` con el nombre:
   - `logo_ROCKET.png` (o el formato que tengas: .png, .jpg, .svg)

2. **Ubicación correcta:**
   ```
   GestorTareasRocke/
     └── static/
         └── logo_ROCKET.png  ← Coloca tu imagen AQUÍ
   ```

3. **Formatos soportados:**
   - PNG con fondo transparente (recomendado)
   - SVG (escalable, mejor calidad)
   - JPG/JPEG (si no tienes los anteriores)

## Importante:
- ✅ La imagen va en `static/` (NO en `templates/`)
- ✅ El nombre debe ser exactamente: `logo_ROCKET.png` (o el formato que tengas)
- ✅ Flask sirve archivos estáticos desde la carpeta `static/`

## Nota:
El código ya está preparado para usar la imagen. Solo necesitas colocar el archivo en la carpeta `static/` con el nombre `logo_ROCKET.png`.
